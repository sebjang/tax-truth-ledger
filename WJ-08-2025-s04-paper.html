// 명제 1 · 소득성격 결정권
소득성격 결정권은 ICI에 있다
DVCA = 지급금액 정보 QI = 세액정산 정보 ICI = 소득성격 정보 ★ 유일 결정권 ROC·배당·§871(k) 구분은 ICI 원장으로만 확정된다

// 명제 3 · QI 기능 한정
QI = 세액원장 · 소득성격 창설 불가
QI = 원천징수·환급·납부세액 ROC·배당·자본이득을 새롭게 정의하지 않는다 ∴ QI = 세액원장
// 명제 4 · KTAX 계승
KTAX는 ICI 소득성격을 계승
DVCA → QI → ICI → KTAX ICI_BOX3(ROC) → 국내에서도 ROC §871(k)(STCG) → 국내에서도 자본이득 ∴ 국내과표 = 국제원장 계승
// Tax Deferral — 과세이연 구조

ROC 비과세 ≠ 영구 면세

STAGE 4
