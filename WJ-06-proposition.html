// 정리 1 · THEOREM_01
ICI 데이터 없이 총분배금(ICI_TOTAL)을 복원할 수 없다
증명. DVCA = 지급사실만 보유.
QI = 세액사실만 보유.
ICI_TOTAL = ICI_BOX1 + ICI_STCG + ICI_BOX3 — 구성요소는 ICI만 제공.
따라서: ICI 부재 → ICI_TOTAL 미확정 → 총분배금 복원 불가.
□
// 정리 2 · THEOREM_02
소득성격(ROC / 배당 / STCG)의 결정권은 오직 ICI에 있다
증명. ROC / 배당 / STCG 구분 = ICI BOX 분류에서만 제공.
DVCA·QI는 금액·세액 정보만 제공 — 소득성격 분류 기능 없음.
따라서: ICI 없이 ROC / 배당 / STCG 구분 불가.
□
// 정리 3 · THEOREM_03
QI 재분류는 세액정산 기능만 수행하며, 소득성격을 결정하지 않는다
증명. QI 재분류 = 원천세 환급 처리.
환급 근거 = ICI가 제공한 BOX3(ROC).
QI가 자체적으로 BOX 분류를 생성하지 않음.
따라서: 소득성격 결정권 ∉ QI. → QI = 세액원장.
□

// §5 · Tax Deferral

ROC 비과세 ≠ 영구 면세
과세이연(Tax Deferral)이다

STAGE 1
ROC 수령
ICI_BOX3 = ROC
과표 제외
비과세 ← 현 시점
원금이 돌아왔을 뿐
STAGE 2
Basis 감소
조정취득원가
= 원가 − 누적ROC
복리 재투자 온전히
과세 이연 중
STAGE 3
Basis 소진
조정취득원가
→ 0
누적ROC ≥ 원취득원가
이연 한계 도달
STAGE 4
Capital Gain 과세
매각가 − 조정취득원가
= Capital Gain → 과세
세원 이연 완성
정확한 과세 실현
// 원금은 원금으로, 이익은 이익으로
ROC 즉시과세 → 이중과세 + 복리 훼손 + 장기 세수 감소
ROC 과세이연 → 투자자 보호 + 복리 온전 + 매각 시 Capital Gain 정확 과세
국제원장에 일치 = 투자자 보호 + 세원 확보 동시 달성
// FINAL UNIFIED PROPOSITION · 최종 통합 명제
소득성격은 ICI가 결정한다
총분배금은 ICI_TOTAL로 복원된다
QI는 세액을 정산한다
KTAX는 국제원장의 소득성격을 계승한다

따라서 국내 과세표준은
국제원장(ICI/QI)과 소득성격이 일치해야 한다

ROC는 국제원장에서 ROC이면 국내원장에서도 ROC다
자본반환은 소득으로 변환될 수 없다
원금은 원금으로 관리하고
이익은 이익으로 과세하라
이것이 자본주의 회계·과세의 기본 원리다
