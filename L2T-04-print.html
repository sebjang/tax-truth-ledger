§1 · 왜 지금인가

72년 격차를 지금 끝내야 합니다

IRC §301(c)(2) 1954년 확립. ICI Primary Layout 1993년 공시 시작. KSD가 2022년 공식 인정: "소득성격 결정권은 ICI에 있다." 그런데 2026년에도 ICI는 국내 세제에 연결되지 않았습니다.

해외 ETF 투자자가 300만명을 넘었습니다. MSTY BOX3=100%, CONY 97%, NVDY 52.6%. 원금이 돌아오는데 배당소득세를 냅니다.

§2 · 5대 개혁 의제

지금 당장 만들 수 있습니다

①
즉시 · 기재부 · 소득세법 §17
소득성격 원장 공개 — ROC 예외조항 신설
"IRC §301(c)(2)에 따라 자본반환으로 확정된 분배금은 배당소득에서 제외하며 취득가액 감소로 처리한다." 문장 하나면 됩니다.
현재: 유권해석 서면-2024-소득-4836 — ICI 검토 여부 불명
개혁 후: 소득세법 명문화 · 납세자 검증 가능
②
즉시 · KSD · 국세청 IT기획과
ICI Primary Layout ↔ 국세청 자동 연결
BOX3 확정 즉시 과표 자동 제외. 원천징수 자동 면제. 경정·환급 자동화.
기술 장벽 없음. KSD 허브 이미 존재.
현재: ICI 데이터 국세청 미연동 · 납세자가 직접 확인·신고
개혁 후: ICI→KSD→국세청 자동 파이프라인
③
6개월 · 금융위 · 자본시장법
과표 감사추적성 의무화 — BOX 3분할
외국납부세액영수증 BOX1(배당) / BOX3(ROC) / STCG(단기양도) 3분할 표기 의무화. 원장의 경계를 영수증에 새깁니다.
현재: 합산 표기 · 소득성격 구분 불가
개혁 후: 3분할 표기 · 납세자 직접 검증 가능
④
2년 · 금융위 · 금투협 · 소득세법 §97
BT(Basis Tracking) 의무화 — 취득원가 추적 체계
LOT 단위 취득원가 추적 시스템 의무 구축. ROC 수령 시 조정취득원가 자동 감액.
미국 2012년 의무화. 한국은 10년 늦었습니다.
현재: 취득원가 추적 없음 · Basis 왜곡
개혁 후: 자동 추적 · 매도 시 정확한 양도세
⑤
1년 · 국회 · 국세기본법 개정

납세자가 자신의 과표 형성 근거 원장을 열람·검증할 권리 명문화.
